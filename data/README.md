
## Data Processing

### Downloaded Datasets

1. Download the datasets from the provided [links](https://drive.google.com/drive/folders/126NTICVEAVj0lpIKYvZGjza-WakYz4I8?usp=sharing).
2. Extract the downloaded files and place them under the directory `data/Processed`.

### Processing Custom Datasets

1. Follow the instructions in [FreeRec](https://github.com/MTandHJ/freerec/blob/master/dataset%20processing.md) to preprocess the datasets.
2. Encode item features using `encode.py`. Please modify the `root` and `dataset` variables, as well as specify the required fields to be encoded.
